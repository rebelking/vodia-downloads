Vodia Pharmacy AI

DNS-enforced installer and full Audio AI script package.

Package Files

  • install-ubuntu.sh
  • install-guide.md
  • audio-ai-script.js
  • NPM package link

HTTPS Install

DNS must point to this server before the installer continues.

curl -fsSL https://get.tryvodia.com/pharmacy-ai/install-ubuntu.sh | sudo env DOMAIN=pharmacytest.audiomercy.com ENABLE_HTTPS=true bash

Generate Audio AI Script Values

Run this only after the HTTPS installer completes successfully.

sudo npx -y vodia-pharmacy-ai@0.2.3 vodia-script --dir /opt/vodia-pharmacy-ai-demo

Only Change These Script Values

var pharmacyApiSecret = "__PHARMACY_API_SECRET__"

var pharmacyRequestUrl = "__PHARMACY_BASE_URL__/api/ai/refill-intake"
var customerLookupUrl = "__PHARMACY_BASE_URL__/api/ai/customer-lookup"
var customerEnrichUrl = "__PHARMACY_BASE_URL__/api/ai/refill-request-enrich"
var fulfillmentUrl = "__PHARMACY_BASE_URL__/api/ai/request-fulfillment"
var pharmacyLocationSearchUrl = "__PHARMACY_BASE_URL__/api/ai/pharmacy-location-search"

var staffTransferDestination = "__STAFF_TRANSFER_DESTINATION__"

var allowDemoPharmacyFallback = true
Do not paste the OpenAI API key into the script. Enter it in the Vodia Audio AI / Voice Agent OpenAI key field.

Production Reminder

For production, set allowDemoPharmacyFallback to false, rotate the secret, remove demo data, and review PHI/HIPAA requirements.
